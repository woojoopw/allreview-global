# ALL Review Global — 구현 계획

> 전세계 어디서든 위치 켜면 주변 장소 리뷰를 한 번에 볼 수 있는 글로벌 리뷰 플랫폼

---

## 핵심 목표

- GPS 켜는 순간 → 그 나라/도시 주변 장소 리뷰 바로 표시
- 리뷰 많은 순 / 점수 높은 순 정렬
- 카테고리별 탐색 (맛집, 카페, 숙소, 관광지 등)
- 12개국 언어 지원

## 데이터 소스

- **Google Places API** — 전세계 장소 + 리뷰 데이터
  - 어느 나라든 GPS 좌표만 있으면 주변 장소 검색 가능
  - 리뷰 수, 평점 데이터 포함
  - 사진 포함

---

## 화면 구성 (총 4개)

### 1. 홈 (/)
- Möbius 로고 + "ALL Review" 브랜딩
- 언어 선택 버튼 (우측 상단)
- 검색창 (키워드 검색)
- "내 주변 탐색" 버튼 (GPS)
- 카테고리 사이드바:
  - 데스크탑: 왼쪽 고정 사이드바
  - 모바일: ≡ 햄버거 버튼 → 왼쪽에서 슬라이드인 드로어
  - 카테고리 항목: 이모지 아이콘 없이 텍스트만

### 2. 검색 결과 (/results)
- 검색어 기반 Google Places 결과
- 리뷰 많은 순 / 점수 높은 순 정렬 탭
- 장소 카드 (이름, 평점, 리뷰 수, 카테고리, 사진)
- 리뷰 없는 장소는 표시 안 함

### 3. 내 주변 탐색 (/explore)
- GPS 위치 기반 주변 장소
- 반경 선택 (500m / 1km / 3km / 5km)
- 카테고리 필터
- 리뷰 많은 순 / 점수 높은 순 정렬

### 4. 장소 상세 (/place/[id])
- 장소 사진
- 평점 + 리뷰 수
- Google Maps 리뷰 목록
- 지도 보기 버튼
